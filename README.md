# 🏥 Clínica Arteaga - Fisioterapia y Rehabilitación

Proyecto web profesional desarrollado para **Clínica Arteaga**, enfocado en la gestión de servicios de rehabilitación física avanzada y bienestar integral.

## 🚀 Características
* **Diseño Moderno:** Interfaz limpia y profesional usando Google Fonts (Outfit).
* **Multi-página:** 5 secciones completas (Inicio, Servicios, Información, Testimonios y FAQ).
* **Interactividad:** Botón flotante de WhatsApp para agendamiento directo.
* **Totalmente Responsivo:** Adaptado para dispositivos móviles y tablets.
* **Optimización de Iconos:** Integración con FontAwesome para redes sociales.

## 🛠️ Tecnologías utilizadas
* HTML5 / CSS3 (Estructura y Estilos personalizados)
* Flexbox y CSS Grid (Layout avanzado)
* FontAwesome (Iconografía)
